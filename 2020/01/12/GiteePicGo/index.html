Gitee + PicGo 个人图床搭建使用说明

1. 简介

本文将详细介绍如何使用 Gitee 和 PicGo 搭建个人图床。通过该图床，你可以方便地上传图片并获取图片链接，用于博客、文档等场景。

2. 准备工作

2.1 注册 Gitee 账号

如果你还没有 Gitee 账号，请先前往 Gitee 注册一个账号。

2.2 创建仓库

  1. 登录 Gitee 后，点击右上角的 + 号，选择 新建仓库。
  2. 填写仓库名称（如 images），选择 公开 或 私有（私有仓库需要配置 SSH 密钥），然后点击 创建。

2.3 生成私人令牌

  1. 点击右上角头像，选择 设置。

  2. 在左侧菜单中选择 私人令牌。

  3. 点击 生成新令牌，勾选 projects 权限，然后点击 提交。

  4. 复制生成的令牌，保存好，后续会用到。

3. 安装 PicGo

3.1 下载 PicGo

前往 PicGo 官方 GitHub 仓库 下载最新版本的 PicGo。

3.2 安装 PicGo

根据你的操作系统，选择相应的安装包进行安装。

4. 配置 PicGo

4.1 打开 PicGo

安装完成后，打开 PicGo。

下载插件

4.2 配置 Gitee 图床

  1. 在 PicGo 左侧菜单中选择 图床设置，然后选择 Gitee。
  2. 填写以下信息：
    • repo: 你的 Gitee 用户名/仓库名，例如 yourname/images。
    • branch: 分支名称，默认为 master。
    • token: 之前生成的私人令牌。
    • path: 图片存储路径，例如 img/。
    • customUrl: 如果你使用了自定义域名，可以填写，否则留空。
  3. 点击 确定 保存配置。

4.3 设置为默认图床

在图床设置页面，将 Gitee 设置为默认图床。

5. 使用 PicGo 上传图片

5.1 上传单张图片

  1. 打开 PicGo，点击 上传区。
  2. 拖拽图片到上传区，或点击 点击上传 选择图片。
  3. 上传成功后，PicGo 会自动将图片链接复制到剪贴板。

5.2 批量上传图片

  1. 打开 PicGo，点击 上传区。
  2. 拖拽多张图片到上传区，或点击 点击上传 选择多张图片。
  3. 上传成功后，PicGo 会自动将所有图片链接复制到剪贴板。

7. 常见问题

7.1 上传失败

  • 检查 repo 是否填写正确。
  • 检查 token 是否有效。
  • 检查网络连接是否正常。

7.2 图片无法访问

  • 检查仓库是否为公开仓库。

  • 检查 customUrl 是否配置正确。

8. 结语

通过以上步骤，你已经成功搭建了基于 Gitee 和 PicGo 的个人图床。现在你可以方便地上传图片并获取链接，用于各种文档和博客中。如果在使用过程中遇到任何问题，可以参考常见问题或查阅相关文档。
