使用 Docker 部署 WordPress 博客

付鹏篚
容器化

2025-02-26 15:30:00

Docker 是一种轻量级的容器化技术，能够帮助我们快速部署和管理应用程序。本文将详细介绍如何使用 Docker 部署一个 WordPress 博客，包括数据库配置、网络设置以及持久化存储。

示例效果：

  • 在文章列表中，左侧会显示缩略图（thumbnail），右侧显示标题、摘要、标签等信息。
  • 如果 top 为 true，文章会显示“置顶”标志。
  • 点击文章后，进入详情页，顶部会显示特色图（feature），正文内容会按照 Markdown 格式渲染。

你可以根据实际需求调整字段和内容，生成更多类似的示例数据。

Ubuntu 系统

sudo apt-get update
sudo apt-get install docker.io

示例数据说明：

  1. title: 文章的标题。
  2. categories: 文章的分类。
  3. tags: 文章的标签，用于分类和搜索。
  4. excerpt: 文章的摘要，显示在文章列表中。
  5. top: 是否置顶，true 表示置顶，false 表示不置顶。
  6. date: 文章的发布日期和时间。
  7. thumbnail: 文章缩略图的 URL，显示在文章列表的左侧。
  8. feature: 文章特色图的 URL，显示在文章详情页的顶部。
  9. 正文内容: 使用 Markdown 格式编写，支持标题、代码块、列表等。
